# Musicians
🚀 О проекте
[Название Проекта] — это полноценный музыкальный веб-сервис, где:

    🎵 Слушатели находят треки, лайкают и комментируют, а также могут ссылки на все платформы, где есть трек;
    🎸 Музыканты загружают свои композиции, обложки и ноты;
    📚 Ноты и табы доступны прямо в карточке песни — удобно для разучивания;
    ❤️ Социалка: лайки, комментарии, избранное.

Проект написан на FastAPI с серверным рендерингом через Jinja2 — быстро, просто и без сложного фронтенда.
